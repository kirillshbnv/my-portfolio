Snake Frenzy - Atari ST (2D Game)

Project Description: A fast-paced arcade game built in C for the Atari ST, using direct bitmap manipulation and raster graphics. Players control a growing snake with keyboard input, navigating hazards like spikes, borders, and self-collision while consuming randomly spawning bugs. The game runs at a 70Hz refresh rate and is being optimized with Motorola 68k assembly.

Date Started: May 2025

Status: In Development

Role: Game Developer (Low-Level Systems)

Technologies Used: C, Bitmap Graphics, Raster Rendering, 68k ASM (planned)

Key Features:

  • Monochrome raster graphics via direct pixel rendering
  • Randomized bug and spike spawning
  • Real-time collision detection at 70Hz refresh rate
  • Planned splash screen with mouse interaction
  • Sound effects for game events
  • Optimized game loop (68k ASM planned)
